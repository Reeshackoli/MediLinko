## ğŸ†˜ Quick Troubleshooting

### App won't build?
```bash
flutter clean
flutter pub get
flutter run
```

### Backend won't start?
- Check MongoDB is running
- Check `.env` file exists and has correct values
- Check `firebase-service-account.json` exists

### Notifications not working?
- Did you run `flutterfire configure`?
- Is `google-services.json` in `android/app/`?
- Is `firebase-service-account.json` in `backend/config/`?
- Check FCM token in database: `db.users.find()`
